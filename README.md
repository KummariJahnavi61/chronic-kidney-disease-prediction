
# Chronic Kidney Disease Prediction

## About
A machine learning project to predict whether a patient 
has Chronic Kidney Disease (CKD) based on 25 clinical features 
collected from 400 patients.

## Dataset
- Source: Kaggle UCI CKD Dataset
- Rows: 400
- Features: 25 numerical and categorical
- Target: CKD / Not CKD
